## 创建新文章
```
    hexo new [layout] <title>
```

## 发布到github
```
    hexo clean //清除之前的东西
    hexo generate //生成静态文章
    hexo deploy // 部署文章
```

## Hexo开启本地站点
```
    hexo s --debug
```

## Hexo创建标签/分类/其他页面
```
    hexo new page tags //新建标签页面
    hexo new page categories //新建分类页面
```
