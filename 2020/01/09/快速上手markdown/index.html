快速上手Markdown

前言

为什么选用Markdown?
Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。

  1. 易于阅读、具有良好的写作体验
  2. 大多数平台的支持，无缝接入
  3. 上手快，基本上看完这篇文章就能快速入门
